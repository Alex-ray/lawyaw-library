## Lawyaw Library

### To Use
clone this repository
`git clone git@github.com:Alex-ray/lawyaw-library.git`

### Navigte into the repository
`cd lawyaw-library`

### Install the dependencies
`yarn`

### Run production
`npm run build && npm run start`

### Run development
`npm run dev`
